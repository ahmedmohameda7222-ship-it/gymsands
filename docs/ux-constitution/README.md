## Required UX model

Plaivra is a persistent fitness context and execution platform for ChatGPT.

- ChatGPT provides reasoning and intelligent execution.
- Plaivra stores, visualizes, tracks, corrects, and protects structured user-owned data.
- Do not create a copy/import queue or a second approval flow after a successful tool-confirmed action.
- Preserve fast direct execution controls when ChatGPT is unnecessary.
- Use task-specific context permissions and clear pending/success/failure states.

## Core route standard

Every route must have:

1. one dominant user job;
2. one clear primary action;
3. a visible current state;
4. loading, empty, error, retry, pending, and success behavior where relevant;
5. correction and recovery paths;
6. accessible target sizes, semantics, focus, contrast, text scaling, and reduced motion;
7. responsive/adaptive behavior defined by the cross-platform constitution.

## Source hierarchy

When old audits or route progress files conflict with the cross-platform constitution, the cross-platform constitution wins.

Completed route audits and progress trackers are not product authority and should not be recreated as permanent source documents.
